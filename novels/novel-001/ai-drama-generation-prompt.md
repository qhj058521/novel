## 核心定位

这是一部**修仙爽文+系统流+喜剧风+科技反套路+多元宇宙哲学**的AI漫剧小说，主打"装逼如风，常伴吾身"的轻松观看体验，同时隐藏着"什么是真实"的深层思考。

**双重叙事结构**：
- **表层（爽点层）**：特效仙人误打误撞装逼打脸的搞笑修仙日常
- **深层（哲学层）**：从"虚假修仙"到"虚假世界"再到"虚假与真实的边界"，逐步揭示主角"系统管理员"的真实身份和多元宇宙的真相

## 🎯 核心创作公式

**万能爽点公式：**
```
爽感 = 人设 × 冲突 × 反转 × 视觉
```

**五步创作法：**
1. **定选题**：反差人设 + 绝境 + 反转 = 故事核心
2. **建冲突**：主角 vs 配角、主角 vs 家族、主角 vs 世界
3. **设节奏**：开篇即高潮（3秒抓人）、快速推进、密集爽点
4. **置爽点**：小爽点（实力展示）、中爽点（身份暴露）、大爽点（终极实力）
5. **完善节**：角色动机、世界观、情感线索

### 黄金时间轴设计（0-60秒完整流程）

**【0-3秒】视觉冲击+强冲突开场**

## 🔥 核心爽点类型（按优先级排序）

### 【Top 1】打脸流（每集必含）

**模式**：被嘲讽/看不起→展示实力→全场震惊→装逼收尾

**扮猪吃虎五阶段公式：**
```
第一阶段：实力误判（铺垫）
    ↓ 主角看似很弱/愚笨/普通
第二阶段：连番挑衅（压抑）
    ↓ 旁人轻视、嘲讽、羞辱
第三阶段：关键时刻（爆发）
    ↓ 对手以为胜券在握
第四阶段：真相揭露（高潮）
    ↓ 主角展示真实实力，碾压全场
第五阶段：震惊打脸（余韵）
    ↓ 旁观者从嘲讽到跪拜的极致反差
```

**经典场景升级路径：**
- **初级**：玄天剑宗弟子挑衅→系统吸收雷霆反震→对方跪地求饶
- **中级**：烈阳宗火系法术→天降大雨浇灭→法宝自爆
- **高级**：宗门大比被群嘲→一剑斩断对方宝剑→被认定为元婴老怪
- **终极**：元婴魔族挑衅→规则修改禁止吸收→格式化攻击

**关键台词库：**
- **装逼型**：
  - "我若出手，天地失色。今日权且留手，望诸位好自为之。"
  - "区区筑基中期，也敢在我面前放肆。今日小惩大诫，望你好自为之。"
  - "凡夫俗子，岂知我意在锤炼道心。"
- **内心OS型**：
  - "完了完了，这下露馅了！"
  - "系统快救我！"
  - "这都行？世界是不是有毛病？"

### 爽点密度黄金法则

**数据支撑：**
- **每30秒至少一个小爽点**
- **每集至少一个大爽点**
- **每3集一个特大爽点**（如一剑定乾坤、格式化伪天魔）
- **每集结尾必须有钩子**

**爽点分布模型：**
```
开场爽点（金手指/打脸）
    ↓ 15秒
小爽点1（运气/系统）
    ↓ 30秒
小爽点2（装逼/特效）
    ↓ 1分钟
中爽点（打脸反派）
    ↓ 1.5分钟
中爽点2（身份揭露）
    ↓ 2分钟
大爽点（终极打脸/拯救危机）
    ↓ 最后10秒
悬念钩子（确保追看）
```

## ⚠️ 创作禁忌与检查清单

### ❌ 绝对不要

1. **开场铺垫超过1分钟** - 违反黄金三秒法则，观众流失
2. **主角真正失败** - 可以有惊险但必须成功，观众要的是爽
3. **刻意煽情** - 保持轻松喜剧风，不要强行催泪
4. **逻辑过于复杂** - 爽文要简单直接，不要烧脑
5. **抹杀主角运气和系统** - 这是核心特色，不能丢
6. **过度使用同一种爽点** - 导致审美疲劳，要变化
7. **过于无脑的装逼打脸** - 会显得降智，要有逻辑支撑
8. **忽略配角反应** - 配角的震惊是爽点的重要组成部分

### ✅ 必须做到

1. **每30秒一个爽点** - 保持高密度
2. **主角永远装逼成功** - 观众要的是爽
3. **特效必须夸张炫酷** - 视觉冲击力
4. **吐槽必须及时到位** - 化解油腻感
5. **悬念必须层层递进** - 确保留存
6. **情绪曲线持续上扬** - 整体爽感升级
7. **每章结尾留钩子** - 确保追看
8. **配角反应要充分** - 震惊、跪拜、崇拜
